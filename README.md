### Complex System Simulation

Usage:
Run for instance:
```
python3 run.py -n 5 -l -r testrun
```
or
```
python3 run.py -a
```

or see help for more information:
```
python3 run.py -h
```

Check requirements.txt for required packages or use the prespecified pipenv.
